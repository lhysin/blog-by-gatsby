파일
```shell
# 파일생성
touch {fileNm}

# 파일명 목록
ls

# 파일 상세 목록
ls -all
```

디렉토리
```shell
# 디렉토리 생성
mkdir {dirNm}

# 현재디렉토리 정보
pwd

# 최상위 디렉토리로 변경
cd /

# home 디렉토리로 변경
cd ~

# 디렉토리 경로 변경
cd {path}

# 상위 디렉토리로 변경
cd ..
```

실행
```shell
# run.sh shell script
bash run.sh
sh run.sh
```
